# Contribution guidelines

Thank you for your interest in improving our project!

Please review the [org-level contribution
guidelines](https://github.com/cipherdothost/.github/blob/trunk/CONTRIBUTING.md)
for high-level guidelines on how to contribute to **Pkgdex**.

In addition to the guidelines above, please keep the following in mind:

- **Dependencies**: Avoid adding new dependencies to the project. If you
  still need to add one and have a good reason to, don't forget to
  update [LICENSE-3rdparty.csv](LICENSE-3rdparty.csv).
- **Documentation**: Any changes or additions made to the API should
  have accompanying documentation.
- **Changelog**: Update the `Unreleased` section of
  [CHANGELOG.md](CHANGELOG.md).
